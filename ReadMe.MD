###CADSR

Environment.yml is a conda version of the environment and requirements.txt is a pip frozen version of the environment.
An example of running CADSR is provided in main.py. To run the method call the line of code bellow after setting up the environment.
 ```
 python main.py
```

We've included one Feynman problem from the SRBench dataset as an example.
A Jupyter notebook is also provide to show some analysis of the outputted datafile by CADSR.

Additionally, all the hyper-parameters for the ablations are supplied in the ablations folder.
